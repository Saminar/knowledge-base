🚀 快速开始
# 1. 在 GitHub 创建 Private 仓库 # 登录 github.com → New repository # Name: knowledge-base / Visibility: Private # 2. 关联远程仓库 cd knowledge-base git remote add origin git@github.com:YOUR_USERNAME/knowledge-base.git # 3. 首次推送 git commit -m "🎉 init: knowledge base scaffold" git push -u origin main # 4. 开启安全功能 # Settings → Code security → 开启所有选项 # 5. 安装 pre-commit（可选） pip install pre-commit pre-commit install # 6. 本地预览文档站点 pip install mkdocs-material mkdocs serve
