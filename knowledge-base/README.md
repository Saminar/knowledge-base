## 🚀 快速开始

```bash
# 克隆仓库
git clone git@github.com:YOUR_USERNAME/knowledge-base.git
cd knowledge-base

# 本地预览（需安装 MkDocs）
pip install mkdocs-material
mkdocs serve
# 访问 http://localhost:8000
```

## 📝 贡献指南

1. 创建新分支：`git checkout -b docs/your-topic`
2. 添加/修改文档
3. 提交 PR，等待 Review
4. 合并到 `main` 分支，自动部署

## 🔐 安全须知

- 本仓库为 **Private** 仓库
- **禁止**提交任何密钥、Token、密码等敏感信息
