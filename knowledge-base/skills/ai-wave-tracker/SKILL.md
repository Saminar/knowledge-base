# 🌊 AI浪潮产业链追踪 Skill

## 方法论：四象限产业透视法

本 Skill 基于以下分析框架，对 AI 产业链进行系统性解构：

```
┌─────────────────────────────────────────────────────────┐
│              AI 浪潮产业链四象限透视法                   │
├───────────────────┬─────────────────────────────────────┤
│  象限一：算力底座  │  象限二：数据通路                   │
│  AI GPU / ASIC    │  光模块 / 交换机 / 网络              │
│  先进代工 / HBM   │  800G → 1.6T → CPO 演进             │
├───────────────────┼─────────────────────────────────────┤
│  象限三：算法平台  │  象限四：应用落地                   │
│  云厂 / 大模型    │  AI Server / AI PC / AI Phone        │
│  国产替代生态     │  边缘推理 / 行业垂直应用             │
└───────────────────┴─────────────────────────────────────┘
```

**五维投资评估框架（每家公司量化打分 0-100）：**
1. **技术壁垒**：核心技术领先度、专利护城河、研发投入
2. **市场地位**：份额、定价权、客户集中度
3. **AI 受益度**：业务与 AI 浪潮的直接绑定程度
4. **政策顺风**：国产替代、大基金加持、出口限制受益
5. **估值安全边际**：PE/PEG、与历史/同行对比

## 完整工作流程

```
用户触发 "AI浪潮分析" / "AI产业链追踪" / "港股AI推荐"
│
├─ Phase 1: 行情数据采集（并行）
│   ├─ fetch_stock_data.py  → /tmp/aiwave_stock.json
│   └─ 覆盖：港股12支 + A股13支 + 美股10支
│
├─ Phase 2: 资讯与催化剂采集
│   ├─ fetch_news_analysis.py → /tmp/aiwave_news.json
│   └─ WebSearch: AI产业最新动态、光模块/GPU行业新闻
│
├─ Phase 3: AI 多维度量化分析
│   ├─ 五维评分（技术/市场/AI受益/政策/估值）
│   ├─ 港股分级推荐（一线/二线/回避）
│   ├─ 产业链联动逻辑梳理
│   └─ 输出 → /tmp/aiwave_analysis.json
│
├─ Phase 4: 可视化 HTML 报告生成
│   ├─ generate_report.sh → /tmp/aiwave_report_YYYY-MM-DD.html
│   └─ 包含 11+ ECharts 图表（暗色主题）
│
├─ Phase 5: 同步到 GitHub knowledge-base（⚠️ 必须！）
│   └─ sync_report.sh → Saminar/knowledge-base/docs/investment/
│
└─ 完成：展示报告路径与核心结论
```

**评分标准（signal字段）：**
| 值 | 含义 | 触发条件 |
|----|------|---------|
| `buy` | 建议买入 | 五维综合分 ≥70 + 近期催化剂明确 |
| `accumulate` | 逢低积累 | 五维综合分 60-70 + 估值合理 |
| `hold` | 持有观望 | 五维综合分 50-60 + 无明显变化 |
| `reduce` | 减仓 | 五维综合分 30-50 或估值过高 |
| `sell` | 建议卖出 | 五维综合分 <30 或基本面恶化 |
